About this Website:

Welcome to Deterrers, the Automated Network Perimeter Threat Prevention System.
This website was developed by Nikolas Wintering (email: nwintering@uos.de)!

Usage:

  1. Login.
  2. Visit your overview page. On first visit you will be asked to choose your department.
  3. Add new hosts. You can either choose yourself as the admin or your whole department.
  4. Choose a host from the list. The list contains all hosts you or your department has been assigned to as administrator.
  5. Several actions can be performed on a host:
    • Register:
      The host will be scanned thoroughly for vulnerabilities and unblocked by the perimeter firewall on success(not supported yet). In case there are security risks, the host remains blocked. (not supported yet)
      Information on the scan results will be send via e-mail to the admin(s).
      Additionally, the host will be scanned periodically.
    • Ordinary Scan:
      The host will be scanned for vulnerabilities but not as thoroughly as with registration.
    • Block:
      Remove host from periodic scan and block the host at the perimeter firewall.(not supported yet)
    • Edit Details:
      Choose a profile for the service the host provides and choose the host-based firewall program running on the host.
      This information is used for configuring the perimeter firewall to only allow traffic for that service.
      This information is also used for generating the host-based firewall configuration script.
      Each time this information is changed the host-based firewall policies are adapted.
    • Edit host-based firewall policies:
      Specify firewall rules for more precise control over the host-based firewall configuration.
    • Download host-based firewall configuration script:
      Generate and download the configuration script.
